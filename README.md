# Google Tools

A collection of miscellaneous tools for working with Google services and APIs.

## Available Tools

### [Chrome Extension ID Resolver](./resolve-chrome-extensions)

A CLI tool to resolve Chrome extension IDs into their official names by scraping the Chrome Web Store.
Useful for identifying extensions in bulk when you only have their IDs.

- **Features**: Batch resolution, CSV export, rate limiting.
- **Prerequisites**: Node.js v18+.
